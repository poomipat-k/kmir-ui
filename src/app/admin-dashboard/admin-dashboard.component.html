Admin IR

หน้านี้พัฒนาเพื่อการใช้งานของนักวิชาการด้านต่างประเทศ สสส. ในฐานะผู้รับผิดชอบในบทบาทหลัก 3 ส่วน ได้แก่ (1) ผู้ประเมินร่วมและบันทึกกิจกรรมต่างๆ ในความก้าวหน้างานด้านต่างประเทศของทุกแผน (2) ผู้ประสานงานกับทุกแผนงานเมื่อถึงรอบการประเมินความก้าวหน้า ตามแต่ที่กำหนดร่วมกัน และ (3) ผู้ดูแลระบบหรือแอดมิน ในการให้อนุญาตแก้ไขข้อมูลต่างๆ รวมทั้งการให้คำแนะนำและแก้ไขปัญหาเบื้องต้นของการใช้งาน


หลักคิดของระบบคือ แอดมินหรือนักวิชาการด้านต่างประเทศ สสส. เป็นผู้ได้รับสิทธิ์ในการเข้าดูเนื้อหาและแก้ไขได้ในทุกส่วน ร่วมกับทุกแผน ในแต่ละแผนจะสามารถเห็นและแก้ไขข้อมูลได้เฉพาะแผนที่ตนเองได้รับสิทธิ์เท่านั้น
